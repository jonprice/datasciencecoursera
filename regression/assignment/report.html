Executive Summary

Exploratory Data Analyses

The data set contails 11 variables, output can be seen in figure 1 of appendex. From this output we can see that Transmission type is coded as a numeric, but really should be a factor. Before correcting this a correlation matrics can be generated, See figure 2 for output.

cor(mtcars)

Correct the transmission type to a factor

mtcars$am <- as.factor(mtcars$am)
levels(mtcars$am) <- c("auto", "man")

Method for model selection

To answer the questions about transmision type on MPG we need to generate a model that will allow the transmission type to be evaluated while holding the other factor equal, i.e. simply comparing the mean MPG of the two transmision types is invalided as transmision type is likly confounded with other variables that influance MPG. Simple creating a model that uses all variables is also not likly the best model as with the small dataset its likly to overfit the data.

Using the correlation tabel focussing on the mpg column the wt (weight) has the highest corraltation to MPG. The next 3 values cyl, disp and hp are also high, but when checking these varables for corralation to wt it can be seen they are also highly corralated and inclustion will increase the model bias. To select futher variables the step function is used to automaticly search for the best model.

mod <- lm(mpg ~ ., data = mtcars)
stepOutput <- step(mod,trace=FALSE)
stepOutput$call
## lm(formula = mpg ~ wt + qsec + am, data = mtcars)

From the output of steps is wt, qsec and am. To confirm the output of steps, we can compare the models mpg ~ am vs mpg ~ am + wt vs mpg ~ am + wt + qsec and mpg ~ am vs mpg ~ am + qsec vs mpg ~ am + qsec + wt

fit1 <- lm(mpg ~ am, data = mtcars)
fit2 <- lm(mpg ~ wt + am , data = mtcars)
fit3 <-lm(mpg ~ wt + qsec + am, data = mtcars)
anova(fit1, fit2,fit3)

In both cases we see that adding both wt and qsec had significant (P < 0.05)effect. Therefor the model that will be used to answer the MPG questions is mpg ~ wt + qsec + am.

Model Elvauation

With 95% confidence, we estimate that switching from automatic to manual while keeping weight and 1/4 second mile time the same will results in a 0.05 to 5.83 increase in MPG. Therefor given a car switching from automatic to manual, which is unlikly to effect weight or 1/4 second mile time is better for MPG, although this difference my be very small.
